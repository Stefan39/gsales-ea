Mit dieser Installations-Routine installieren Sie gSales-EA (Einnahme/Ausgabe). Da diese Version immer noch im BETA-Status ist, geschieht die Installation auf eigene Gefahr. Wir übernehmen keine Haftung bei Schäden an Ihrem System und/oder ihrer vorhandenen gSales-v2 Installation.

gSales-EA liest eigentlich zwar nur von gSales-v2, kann aber nicht garantieren, dass bei der einen oder anderen API-Funktion von gSales nicht auch Daten in einer gSales-v2-Version verändert werden.

Alle Ihre hier hinterlegten empfindlichen Daten werden im Verzeichnis /includes/config und im Vereichnis /pdfs gespeichert. In beiden Verzeichnissen sollte eine Datei namens .htaccess vorliegen, in der lediglich folgendes steht:
Deny from all
Prüfen Sie, ob Dateien in den Ordnern direkt von Ihrer Domain aufrufbar sind:
{$smarty.server.HTTP_HOST}/includes/index.php
Gibt der Browser bei Eingabe der obigen URL den Text "Oh oh... nicht gut" aus, funktioniert die Direktive nicht. Sprechen Sie in diesem Fall Ihren Server-Admin an, der dafür Sorge tragen sollte, dass diese beiden Verzeichnisse geschützt sind.

Übrigens, dieser Schutz sollte dennoch vorgenommen werden, auch wenn ein separater htaccess AuthBasic Autentifikations-Schutz installiert wird!
